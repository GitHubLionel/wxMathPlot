Detailed Description

Abstract base class providing plot and labeling functionality for functions F:Y->X.


Override mpFY::GetX to implement a function.
Override mpFY::GetMinX and mpFY::GetMaxX to provide min and max X range
Optionally implement a constructor and pass a name (label) and a label alignment to the constructor mpFY::mpFY. If the layer name is empty, no label will be plotted.
